PERGUNTAS FREQUENTES

O QUE PRECISA PARA ANUNCIAR❓
✔️ Ser inscrito no grupo. ✔️ Ter um @ nome de usuário. ✔️ Clicar em: FAZER CADASTRO. ✔️ Clicar em: FAZER ANÚNCIO. ✔️ Preencher o FORMULÁRIO. ✔️ No formulário Clique: ENVIAR.
O QUE É O CADASTRO❓
✔️ É a coleta de informações via Bot. ✔️ Usamos as informações nos botões. ✔️ Garantindo a execução correta.
O QUE É O FORMULÁRIO❓
✔️ É um formulário para criar o anúncio. ✔️ Será feita perguntas sobre o anúncio. ✔️ Complete o formulário para enviar. ✔️ O anúncio será enviado ao grupo.
QUAL A MINHA SENHA DO SITE❓
✔️ A senha é entregue ao se cadastrar. ✔️ Clique em: FAZER CADASTRO. ✔️ Não usa senha se entrar pelo grupo. ✔️ Clique em FAZER ANÚNCIO. ✔️ O login é automático pelo grupo.
COMO CRIAR O NOME DE USUÁRIO❓
✔️ Vá nas opções do Telegram. ✔️ Clique em: "Configurações". ✔️ Clique em: "Nome de Usuário". ✔️ Digite um @ Nome.
QUAL A DURAÇÃO DOS ANÚNCIOS❓
✔️ Os anúncios se repetem semanalmente. ✔️ Com duração máximo de um mês. ✔️ Anúncio completo, clique em EXCLUIR.
QUAL A DURAÇÃO DAS MENSAGENS❓
✔️ Mensagens de usuários, duram 9h. ✔️ Mensagens do Bot, duram 5 min.
ERROU A INFORMAÇÃO DO ANÚNCIO❓
✔️ Clique em EXCLUIR e refaça o anúncio. ✔️ Ou, peça ajuda a um Administrador.
COMO REMOVER UM ANÚNCIO❓
✔️ Clique em: ❌ EXCLUIR. ✔️ Após Excluir, não se repetem. ✔️ Apenas o Dono consegue excluir.
TEM LIMITE PARA OS ANÚNCIOS❓
✔️ Anúncios diferentes sem limites. ✔️ Se repetir um anúncio, ele se exclui.
APARECEU ANÚNCIO BLOQUEADO❓
✔️ O usuário pode está bloqueado. ✔️ Peça ajuda ao administrador.
ACHOU UM ANÚNCIO SUSPEITO❓
✔️ Você deve fazer a denuncia no grupo. ✔️ O anúncio será excluído. ✔️ O dono do anúncio será investigado. ✔️ O dono do anúncio pode ser expulso.
NÃO ENCONTROU NO FORMULÁRIO❓
✔️ Solicitar no SUPORTE TÉCNICO. ✔️ Enviar mensagem para @admin. ✔️ Aguarde uma resposta.
MUDOU DE @ NOME DE USUÁRIO❓
✔️ Clique em: FAZER CADASTRO. ✔️ O Bot vai atualizar para você.
SAIU DO GRUPO OU APAGOU A CONTA❓
✔️ O Bot vai excluir os seus anúncios.
SEU ANÚNCIO NÃO APARECEU❓
✔️ Refaça o anúncio no formulário. ✔️ Avise sobre o erro se não resolver.
QUAIS AS FORMAS DE PAGAMENTO❓
✔️ Disponível via KOTAS e PIX. ✔️ Variando de MENSAL até ANUAL.
O QUE É O KOTAS❓
✔️ Site para divisão de contas. ✔️ Junte amigos e economize. ✔️ Praticidade e segurança.
COMO DIVULGAR GRUPO KOTAS❓
✔️ No formulário, escolha KOTAS. ✔️ Em grupo KOTAS, digite o Código. ✔️ O código é os 6 números finais do link.
QUAIS OS STATUS DE ASSINATURA❓
✔️ Aguardando - AINDA VAI ASSINAR. ✔️ Assinado - ACESSO IMEDIATO. ✔️ Reservando - ALGUÉM VAI SAIR.
COMO ACESSAR A CONTA❓
✔️ Login e Senha - E-MAIL E SENHA. ✔️ Ativar por código - CÓDIGO NA TV. ✔️ Convite por E-mail - LINK NO E-MAIL.
O QUE É O MENU NOS ANÚNCIOS❓
✔️ São botões de acesso rápido. ✔️ Tem um MENU em cada anúncio. ✔️ Só precisa clicar nos emojis.
O QUE SIGNIFICA OS ÍCONES❓
❤️ Número Total de Curtidas. 💬 Número de Conversas. 📢 Número de Anúncios. 👍 Curtida do Anúncio. ❓ Perguntas Frequentes. 🔍 Pesquisar Contas. 🛠️ Suporte Técnico. ✅ Regras do Grupo.
COMO PESQUISAR NO GRUPO❓
✔️ STREAMING: "/pesquisar Netflix". ✔️ ANUNCIANTES: "/pesquisar @nome". ✔️ O resultado aparece em lista. ✔️ Clique sobre os nomes para acessar. ✔️ Também pode só perguntar no chat. ✔️ O Bot mostrará o botão de pesquisa. ✔️ Também pode usar o NOSSO SITE
O QUE FAZER ANTES DO PIX❓
✔️ Evite pessoas que vem no privado. ✔️ Não aceite nomes diferentes no PIX. ✔️ Peça referências e informações. ✔️ Peça comprovante de conta. ✔️ Verifique os dados da conta. ✔️ Tire todas as suas dúvidas.
O QUE PRECISO VERIFICAR❓
✔️ Se o plano da conta está certo. ✔️ Se está na versão de teste grátis. ✔️ O nome do assinante na conta. ✔️ A quantidade de perfis ativos. ✔️ O tempo que a conta foi feita.
COMO FAZER UMA DENÚNCIA❓
✔️ Enviar mensagem ao @admin. ✔️ Clique em: SUPORTE TÉCNICO. ✔️ Clique em [🛠] no MENU.
O QUE SÃO AS REGRAS DO GRUPO❓
✔️ Lista de coisas PROIBIDAS. ✔️ Lista de coisas PERMITIDAS.
AONDE ESTÁ AS REGRAS DO GRUPO❓
✔️ Digite e envie: "/regras". ✔️ Clique em: MOSTRAR REGRAS. ✔️ Clique em: [ ✅ ] no MENU.
AONDE ESTÁ O LINK DO GRUPO❓
✔️ Digite e envie: "/link". ✔️ Copie o link que aparece.
O QUE É O MODO NOTURNO❓
✔️ Período de descanso do grupo. ✔️ Horário vai de 22hs até 8hs. ✔️ Nesse modo, as mensagens se excluem.
QUAIS MENSAGENS SÃO EXCLUIDAS❓
✔️ Mensagens com links. ✔️ Mensagens com preços. ✔️ Mensagens com palavrões.
COMO ENVIAR MENSAGENS AO ADM❓
✔️ Envie: @admin + Sua Mensagem. ✔️ A mensagem será enviada pelo bot. ✔️ Os Admins receberão a mensagem.
AINDA ESTÁ COM DÚVIDAS❓
✔️ Acesse o SUPORTE TÉCNICO. ✔️ Faça a sua pergunta e aguarde. ✔️ Ou envie mensagem ao @admin.
